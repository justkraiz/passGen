=== Генератор паролей ===

1. Убедитесь, что Java 8 или выше установлена на вашем компьютере.
2. Подготовьте файл символов из которых будут составлены различные комбинации паролей и разместите в текущей папке, например `symbols.txt` с набором символов для генерации.
3. Выполните команду в терминале:
   ./passGen [длина пароля] [файл символов] [выходной файл]

Пример:
   ./passGen 4 symbols.txt result.txt
